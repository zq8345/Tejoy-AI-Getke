## 设计主线

一句话：**消灭"这个按钮跟我现在在哪一页有什么关系"这个问题。**

- 左栏 = 你在哪一格（唯一状态真源）
- 阶段页 = 这一格的**出口动作**
- 每格工具栏回答的是「这一格靠什么排空」，不是「这页能干什么」

每页只回答一个问题：

| 页面 | 唯一问题 | 禁止 |
|---|---|---|
| 待办事项 | 今天先干哪一件？ | 不做动作，只分诊+跳转 |
| 找客户 | 要不要再进货？进多少？ | 不管进来之后的事 |
| 数据看板 | 系统健康吗？哪里漏水？ | 不告诉你今天干嘛 |
| 详情页 | 这一家，下一步做什么？ | 不平铺所有动作 |
